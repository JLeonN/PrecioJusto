// turbo-all

1. Diagnostics: `git status; git diff; git log --oneline -5`
2. Stage: `git add -A`
3. Commit: `git commit -m "Título" -m "Descripción"`
4. Verify: `git status`
5. Finish: Responde ÚNICAMENTE: **Commit listo.**
